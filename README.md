# Static UI - Message Picture

A lightweight and interactive web application buitl with HTML, CSS, and vanilla Javascript.
The UI displays a welcome message and two buttons that let the user change or preset the message.
No server or installation required.

# How to Use

1. Download or clone the project
2. Open index.html in any browser
